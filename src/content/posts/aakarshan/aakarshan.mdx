KMIT's Aakarshan (Artclub) Web Dashboard enables every student to upload their art and inspire all the students this website has features like likes and comments on each others posts , customisability of the posts by the owner and a profile with their profile pic
Dynamic Progressive Web Application build using Angular 8
Implemented Features:
User authentication,
Comments, Likes, Compressing Images client-side, offline capability

Database Design in Google Firebase Firestore
Fully Functional
